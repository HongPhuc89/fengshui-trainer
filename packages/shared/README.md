# @quiz-game/shared

Shared types, interfaces, and DTOs for Quiz Game monorepo.

## Purpose

This package contains TypeScript definitions shared across all applications:

- DTOs (Data Transfer Objects)
- Interfaces
- Type definitions
- Enums

## Usage

```typescript
import { UserDto, BookDto } from '@quiz-game/shared';
```
